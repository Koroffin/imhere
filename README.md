# Поле того, как сделал пул (к тебе пришли новые данные):
Запускаешь у себя в терминале (ты должен находиться в папке imhere):

```sh
npm install
grunt start
```

Первая команда скачивает новые библиотеки, вторая - запускает локальный сервер
